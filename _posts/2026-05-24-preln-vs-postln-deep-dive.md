## 一、结构差异（一句话记住）

| | PreLN | PostLN |
|---|---|---|
| **结构** | `x_{l+1} = x_l + f(LN(x_l))` | `x_{l+1} = LN(x_l + f(x_l))` |
| **比喻** | 高速公路 + 36 条匝道 | 盘山公路 36 弯 |
| **前向** | 累加器（shortcut 直达） | 复合函数（必须经过每层） |
| **反向** | 展开后是累加（有直达项 I） | 纯粹的矩阵连乘 |
| **输出范数** | 线性增长 `O(L)` | 稳定（每层压回 `sqrt(d)`） |
| **梯度** | 稳定但同质化 | 爆炸或消失 |

PreLN 把 LayerNorm 放在子层**之前**，PostLN 放在子层**之后**。这个位置差异看似微小，却从根本上改变了梯度传播的数学结构。

---

## 二、极简 2 层网络：把公式摊开看

### 2.1 PreLN 前向传播

```
x1 = x0 + f(LN(x0))
x2 = x1 + f(LN(x1))
```

**展开 x2：**

```
x2 = x0 + [f(LN(x0))] + [f(LN(x1))]
     ↑主路    ↑匝道0          ↑匝道1
```

**关键观察**：`x0` 直接出现在 `x2` 里！没有经过任何变换。这就是 **shortcut**。

**结果**：输出范数随层数线性增长（36 层 ≈ 6 倍），但这不是致命问题。

---

### 2.2 PreLN 反向传播（核心中的核心）

对 `x0` 求导，先用链式法则：

```
∂x2/∂x0 = (∂x2/∂x1) · (∂x1/∂x0)
```

代入每层局部 Jacobian：

```
∂x2/∂x1 = I + A1    (A1 = ∂f(LN(x1))/∂x1)
∂x1/∂x0 = I + A0    (A0 = ∂f(LN(x0))/∂x0)
```

**最关键的一步——展开括号：**

```
(I + A1)(I + A0) = I + A0 + A1 + A1·A0
```

乘以损失梯度 `g = ∂L/∂x2`：

```
∂L/∂x0 = g·I + g·A0 + g·A1 + g·A1·A0
         ↑      ↑       ↑       ↑
       直达   匝道0   匝道1   高阶小量(忽略)
```

**结论**：

- **第一项 `g·I`**：损失函数的梯度通过单位矩阵 `I` **直接传回 x0**，不经过任何中间层！
- 后面几项是小量修正（`A` 被 LN 和初始化控制得很小）。
- **这不是连乘，是累加！** 因为有 `I` 项保底，梯度不会指数爆炸。

---

### 2.3 PostLN 反向传播

结构：

```
x1 = LN(x0 + f(x0))
x2 = LN(x1 + f(x1))
```

Jacobian：

```
∂x2/∂x1 = J_LN · (I + W1)   ← 没有 I 项！
∂x1/∂x0 = J_LN · (I + W0)   ← 没有 I 项！
```

反向传播：

```
∂L/∂x0 = g · [J_LN·(I+W1)] · [J_LN·(I+W0)]
         ↑    ↑矩阵1         ↑矩阵2
       纯粹的矩阵连乘！
```

**结论**：

- 没有 `I` 项，没有 shortcut，无处可逃。
- 每层 Jacobian 在某些方向上可能放大 1.5 倍，36 层后 `1.5^36 ≈ 2.5×10^6`。
- **梯度必然爆炸或消失。**

---

## 三、LayerNorm Jacobian 的几何本质

LN 不是简单的"减均值除标准差"，它的 Jacobian 是一个**投影矩阵**。

### 3.1 显式公式

对输入 `x ∈ R^d`，LN 输出 `y`：

```
y_i = (x_i - μ) / σ
```

Jacobian 矩阵 `B = ∂y/∂x`：

```
B = (1/σ) · [ I - (1/d)·11^T - (1/d)·yy^T ]
```

其中：

- `I`：单位矩阵
- `11^T`：全 1 矩阵（均值杀手）
- `yy^T`：LN 输出的外积（标准差杀手）

### 3.2 特征值分析（以 d=3 为例）

| 特征向量 | 特征值 | 物理意义 |
|---|---|---|
| `1 = [1,1,1]^T`（全 1 方向） | **0** | 均值方向：同加同减，LN 无视 |
| `y`（LN 输出本身） | **0** | 标准差方向：同比例缩放，LN 无视 |
| 任意 `v ⊥ 1, v ⊥ y` | ≈ `1/σ` ≈ **1** | 真正的信息方向：相对比例变化 |

**核心理解**：

- LN 的 Jacobian 把 `d` 维空间**投影**到一个 `(d-2)` 维子空间上。
- 它杀死了两个无关方向（均值和标准差），保留相对比例方向。
- 在有效子空间上，特征值 ≈ 1，所以 LN 不是"缩小方差"，而是"剪掉无用自由度"。

## 四、Representation Dilution（表征稀释）的本质

### 4.1 不是"梯度消失"，而是"人多势众的淹没"

PreLN 前向展开：

```
xL = x0 + f0(LN(x0)) + f1(LN(x1)) + ... + f_{L-1}(LN(x_{L-1}))
```

假设每层贡献范数都是 `C`：

- 第 0 层贡献占总输出的比例：`C / (L·C) = 1/L`
- 第 30 层贡献占总输出的比例：也是 `1/L`

**绝对贡献没有递减，但相对占比被淹没了。**

类比：36 个人往同一个桶里倒水，每人倒 1 升。第 30 个人倒的水量和第 1 个人一样，但桶里已经有 29 升了，他只贡献了 1/30。

### 4.2 梯度同质化（趋同）

PreLN 第 `l` 层收到的梯度：

```
∂L/∂x_l ≈ g + 小量修正
```

**所有层共享同一个主信号 `g`！**

- 第 1 层听到："让任务成功率更高！"
- 第 30 层听到："让任务成功率更高！"

第 30 层本应该学到"当接触力 > 0.5N 时旋转手腕"，但它收到的指令和第 1 层一样笼统。所以深层只能重复浅层已经会的东西——**趋同**。

> **实证支持**：Scaling Laws (Kaplan 2020) 发现 PreLN 模型中，最后几层的权重更新幅度比前几层小一个数量级——正是 1/L 稀释效应的直接后果。深层在"免费搭便车"，参数几乎不更新。

## 五、PostLN 为什么有天然分工？

PostLN 前向是**复合函数**：

```
x2 = f1(f0(x0))
```

反向传播时，损失函数的指令往回传，每层的 Jacobian 都会对它做变换：

- 第 36 层听到："让成功率更高！"
- 第 35 层听到的是经过第 36 层加工后的："让成功率更高，特别是修正末端执行器姿态"
- 第 30 层听到的是经过后面 6 层加工后的："让成功率更高，特别是处理接触力突变时的旋转策略"
- 第 1 层听到的是经过后面 35 层加工后的："让成功率更高，特别是提取图像边缘特征用于粗定位"

**结果**：每层听到的是被后面层过滤后的、专属于自己的精细指令。深层负责精细操作，浅层负责粗粒度感知——**分工明确**。

代价：36 个加工步骤连乘，梯度必然爆炸。

## 六、残差注意力在这个框架下的位置

**问题**：PreLN 的深层被稀释了（每人只贡献 1/36），而且所有层都在听同一个广播（同质化）。怎么让深层既能享受 PreLN 的梯度稳定性，又能学到独特的东西？

**残差注意力的方案**：

在每一层引入跨层残差连接：

```
A^{(l)} = softmax(QK^T/sqrt(d)) + α · A^{(l-1)}
```

**本质**：给深层单独修了一条"**专用匝道**"——

- 即使第 30 层的新增注意力趋向均匀（被稀释），加上第 1 层的非均匀注意力模式，整体仍保持位置特异性。
- 浅层已经验证有效的注意力模式，通过残差连接直接传递到深层，深层不需要从零学习。
- 深层收到的梯度虽然主体仍是 `g`，但残差项提供了一个"差异化通道"，让深层能专注于"在浅层基础上做精细修正"。

**关于 α 的选择**：

- `α = 0`：退化成标准 PreLN（每层独立）
- `α → 1`：深层几乎复制浅层（过度继承）
- 实际最优通常在 0.1–0.3，让深层"借鉴但不抄袭"

**一句话**：残差注意力在 PreLN 的"并联广播"体系下，给深层发了一张"专用通行证"，让它不必和其他 35 层一起趋同。

## 七、补充：PostLN 的 Jacobian 细节

PostLN Jacobian 严格写应为 `J_LN · (I + ∂f/∂x)`，其中 `∂f/∂x` 不一定是权重矩阵 `W`（因为有非线性激活）。

- **注意力层的线性部分**：`∂f/∂x` 确实就是 `W`
- **FFN 层**：`∂f/∂x = W2 · diag(σ'(W1x+b)) · W1`，这会引入额外的条件数膨胀，让 PostLN 梯度问题更严重

这也是为什么 PostLN 在深层 FFN 模型中尤其难以训练的原因之一。

---

## 八、实践中的中间路线：Sandwich LN

很多现代模型（如 CogView、部分 MoE 模型）采用 **Sandwich LN**：在 PreLN 基础上，`f()` 内部也加 LN。

```
x_{l+1} = x_l + f(LN_out(LN_in(x_l)))
```

相当于给每层的"匝道"加了限速器——既保持 PreLN 的梯度稳定性，又控制了层间方差累积。

## 九、快速记忆卡

| 问题 | PreLN | PostLN |
|---|---|---|
| 前向结构 | `x + f(LN(x))` | `LN(x + f(x))` |
| 输出范数 | 线性增长 | 稳定 |
| 反向 Jacobian | `I + A`（有直达项） | `J_LN·(I+W)`（无直达项） |
| 梯度传播 | 展开后累加 | 矩阵连乘 |
| 深层梯度 | 稳定但同质化 | 爆炸/消失 |
| 层间分工 | 无（并行听广播） | 有（串联流水线） |
| 有效深度 | 稀释（1/L） | 不稀释 |
| 训练难度 | 容易训练但深层废掉 | 难以训练但理论分工好 |

---

## 十、一句话总结

> **PreLN 是"并联广播"**：损失函数直接对每一层喊话，所有人听到的都是同一个笼统指令。梯度通过 `I` 高速公路直达，稳定不爆炸；但 36 层在做重复劳动，深层被淹没（Representation Dilution）。
>
> **PostLN 是"串联流水线"**：损失函数的指令必须经过后面所有层的加工才能传到前面。每层听到的是专属精细指令，分工天然合理；但 36 个 Jacobian 连乘，梯度必然爆炸或消失。
>
> **残差注意力**：在 PreLN 的高速公路上，给深层修了一条"专用匝道"，让它能直接继承浅层有效的注意力模式，从而获得"并联的稳定性 + 串联的分工感"。
