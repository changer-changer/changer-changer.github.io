## 二、字节跳动人才计划详解

### 2.1 Top Seed大模型顶尖人才计划

**基本信息：**
- 启动时间：2024年5月发起，2025年4月正式启动2026届校招
- 招募规模：约30位顶尖应届博士
- 研究课题：大语言模型、机器学习算法和系统、多模态生成、多模态理解、语音等

**申请要求：**
- **学历要求**：2026届应届博士生（严格限制）
- **专业背景**：不限专业，更关注研究潜力
- **核心要求**：
  - 具有极强技术信仰与热情
  - 具备出色研究能力
  - 富有好奇心和驱动力
  - 绝大部分入选者发表过高质量论文或有代表性开源工作
- **选拔方式**：邀约制为主，HR按论文作者名单打电话邀约

**真实门槛分析：**
- 本科生可行性：**几乎为零**
- 实际入选者标签：一流高校硕士/博士生、顶级学霸、IMO满分选手、顶会一作作者
- 项目宣传："期望找大模型领域前5%的人，做95%的人做不到的事"

**薪酬待遇：**
- 年薪：150-200万+
- 职级：3-1起
- 其他：匹配研究价值的回报、一流科研环境、充分自由度

**本科生策略：**
无法直接申请。但可先申字节**研究实习生（Research Intern）**，表现优异后内部推荐。需有顶会论文在投/发表。

### 2.2 筋斗云人才计划

**基本信息：**
- 启动时间：2024年7月首次启动，2025年5月升级
- 职位类型：全职+实习
- 课题数量：2025年增至42个（增加30%）

**申请要求：**
- **全职岗位**：2022年9月-2026年8月期间毕业的博士生
- **实习岗位**：2025年9月及以后毕业的在校博士生
- **专业限制**：不限专业
- **技术领域**：大模型应用、搜索/推荐/广告、计算机体系结构与系统优化、安全/AI Safety、硬件、AI Coding、视频架构、AIGC等8大领域
- **重点针对**：有亮眼学术成果、国际知名竞赛经历、极强实战能力、参与过重要项目

**真实门槛分析：**
- 本科生可行性：**极低**（明确面向博士）
- 2025年3月首次开放实习专项，仅面向博士
- 薪酬：不设上限，日薪可达800-2000元

**薪酬待遇：**
- 薪酬不设上限
- 晋升不看资历
- 行业TOP级算力和数据资源支持
- 技术专家带教

**本科生策略：**
本科生直接申请筋斗云几乎不可能。但可以先申字节**ByteIntern（常规实习生）**，日薪200-500元，表现好可转正。

### 2.3 ByteIntern（常规实习生）

**基本信息：**
- 面向人群：2026届/2027届毕业生（本科、硕士均可）
- 特点：提供转正机会

**申请要求：**
- 本科及以上学历在读
- 计算机、软件工程等相关专业优先
- 热爱编程，逻辑思维强
- 熟练掌握至少一门编程语言（Java/Kotlin/Objective-C/Swift/C/C++/JS/Python）

**薪酬待遇：**
- 日薪：200-500元/天（根据岗位和地点）
- 算法岗可达400-800元/天

**本科生友好度：**⭐⭐⭐⭐⭐（强烈推荐）

**申请方式：**
- 官网投递：https://jobs.bytedance.com/campus
- 全年滚动招聘

## 三、华为人才计划详解

### 3.1 天才少年计划

**基本信息：**
- 发起人：任正非于2019年发起
- 目标：用顶级挑战和顶级薪酬吸引顶尖人才
- 年薪档次：三档（89.6-100.8万、140.5-156.5万、182-201万）

**申请要求：**
- **学历要求**：不限学历、不限学校、不限专业
- **核心要求**：只要有"硬实力"即可
- **实际门槛**：
  - 国际顶级竞赛奖项（ACM-ICPC世界总决赛级别）
  - 或顶会一作论文
  - 或重大科研成果

**真实入选案例分析：**

**案例1：丁聪（西安交大2020届本科）**
- 背景：钱学森书院计算机试验班
- 成就：ACM队骨干，第43届ACM-ICPC全球第41名（刷新交大历史最好成绩）
- 年薪：89.6-100.8万档
- 入职：华为消费者BG鸿蒙操作系统团队

**案例2：张子杰（西安交大2020届本科）**
- 背景：少年班出身，ACM队成员
- 成就：ACM-ICPC亚洲区赛金奖、CCPC银奖
- 年薪：89.6-100.8万档
- 入职：华为消费者BG高斯数据库团队

**案例3：武敏颜（华科2021届本科）**
- 背景：计算机学院
- 成就：综合能力极强，多段名企实习（谷歌、蚂蚁金服、微软中国）
- 特点：非纯技术竞赛路线
- 年薪：超百万

**选拔流程（7轮考核）：**
1. 简历筛选
2. 笔试
3. 初次面试
4. 主管面试
5. 若干部长面试
6. 总裁面试
7. HR面试

**本科生可行性分析：**
- 历史上仅**3名本科生**入选（丁聪、张子杰、武敏颜）
- 核心要求：ACM-ICPC World Finals级别奖项或顶会一作论文
- 普通985/211本科生几乎不可能
- 如果你是大一/大二，目标是ACM-ICPC金牌；如果是大三/大四，基本无望

**薪酬待遇：**
- 本科生档：89.6-100.8万/年（第二档）
- 博士最高档：182-201万/年

### 3.2 全球顶尖AI人才招募计划

**基本信息：**
- 发布时间：2025年10月
- 目标：打造世界一流的AI战队，构建领先世界的大模型
- 负责人：余承东（华为常务董事、终端BG董事长、IRB主任）

**申请要求：**
- **学历要求**：
  - 本硕：2026/1/1-2026/12/31毕业的国内高校本科生与硕士研究生
  - 博士：2025/1/1-2026/12/31毕业的国内高校博士生与海外高校本硕博
- **三大核心标准**：
  1. **学术先锋**：成绩优异，扎实数理功底，重量级科研成果/论文/专利，国际顶级竞赛获奖
  2. **技术热情**：对AI领域怀揣坚定技术信仰与热忱，有志成为技术领军人物
  3. **创新思维**：勇于创新突破，精于洞察，穿透技术本质并付诸实践

**岗位方向：**
- 本硕：AI软件工程师、AI算法工程师、AI数据工程师、AI安全工程师、AI应用工程师、AI Infra工程师
- 博士：AI算法、大模型算法、多模态算法、大模型应用、AI软件开发、AI Infra、AI安全/隐私保护、计算机视觉、NLP/语音语义、机器学习

**工作地点：**
深圳、北京、上海、东莞、南京、武汉、杭州、西安

**薪酬待遇：**
- 行业内具有竞争力的薪酬
- 攻克全球前沿AI技术难题的机会
- 参与打造世界领先大模型项目
- 与图灵奖、菲尔兹奖获得者共事
- 充足的算力资源自由调度

**本科生可行性：**⭐⭐⭐⭐（可尝试）
- **明确面向本科生开放**
- 要求成绩优异+竞赛获奖/科研成果
- 北邮本科生如果成绩好+有竞赛奖项，可尝试申请

**申请方式：**
- 华为招聘官网：https://career.huawei.com
- 或直接投递简历

### 3.3 勇敢星实习生计划

**基本信息：**
- 面向人群：海内外在校优秀大学生（本科、硕士、博士均可）
- 特点：华为官方"勇敢星"品牌，寓意勇敢探索未来之星

**申请要求：**
- 2026届应届生为主
- 计算机/AI相关专业
- 有编程基础
- 无硬性论文/竞赛要求

**申请流程：**
1. 官网投递简历
2. 机考（2小时编程题，偏实际场景）
3. 专业面试（1小时，深挖项目+手撕代码）
4. 业务主管面试（15分钟，考察综合素质）

**真实面经分享：**
- 机考："比较偏实际，总体不难"
- 专业面试：深挖项目和实习经历，手撕代码题（如求解直角三角形数）
- 主管面试：问家庭情况、对华为的了解、工作压力等

**薪酬待遇：**
- 实习期：约7K-8K/月
- 转正后：8K-10K/月（13级）
- 优秀本科生可定14级（15-18K/月）

**转正机会：**
- 优秀实习生可获得校招offer
- 参加过实习的机试和综测成绩在正式校招时仍有效

**工作地：**
南京、东莞、西安、上海

**本科生友好度：**⭐⭐⭐⭐⭐（强烈推荐）
- 这是最容易进入的华为"名头计划"
- 华为实习生普通招聘叫"实习生"，但"勇敢星"是官方包装的人才品牌
- 你的电子设计竞赛经历+机器人项目经验非常适合AI工程师或算法工程师岗位

**申请方式：**
- 华为招聘官网：https://career.huawei.com/reccampportal

## 四、腾讯人才计划详解

**基本信息：**

- 官方面向本硕博，但实际以硕博为主
- 本科生需有顶级竞赛奖项或论文才有机会

**申请方式：**
- 官网：https://join.qq.com/qingyun.html
- 邮箱：txqy@tencent.com

### 4.2 犀牛鸟精英人才计划（精英营）⭐⭐⭐⭐⭐

**基本信息：**
- 启动时间：2017年首次发布
- 目标：选拔对科学研究有热情、有潜力的学生，培养支撑科技自主创新的中坚力量
- 已累计培养海内外410+名科研精英

**申请要求（2026年度）：**
- **面向人群**：海内外高校在校本科生（大三及以上）、硕士及博士生，包含2026年入学新生
- **以下条件至少满足一项**：
  1. 以第一作者（或导师一作、自己二作）在CCF-A类、B类或同等顶级刊物/会议发表论文
  2. 作为骨干参与实验室或导师的省级及以上大型科研项目
  3. 获得国家级学术相关奖项、学生竞赛奖项，或IEEE、ACM、中国电子学会、CCF等学术组织的学生奖项

**优先条件：**
- 曾参与腾讯科研合作项目（CCF-腾讯犀牛鸟基金、专项及联合实验室）
- 曾作为腾讯高校创新俱乐部、星推官等社团核心成员
- 曾获得腾讯相关竞赛奖项（腾讯广告算法大赛、腾讯犀牛鸟开源人才培养计划）

**真实入选案例：**

**案例1：王春实（桂林电子科技大学2021级本科）**
- 成就：以第一作者在IEEE JBHI、Pattern Recognition Letters等发表高水平论文7篇
- 保研至浙江大学
- 入选2025年度腾讯犀牛鸟精英人才计划

**案例2：林泽宇（某985本科）**
- 分享："985研究生占了近全部，我是TARS项目唯一入选的本科生"
- 需完成导师实战任务+源码剖析+编程实战

**培养内容：**

### 10.3 时间规划建议

**如果你现在大一/大二：**
- 目标：积累项目经验和竞赛奖项
- 行动：
  1. 参加腾讯犀牛鸟开源计划（每年6-9月）
  2. 参加电子设计竞赛、ACM竞赛等
  3. 争取发表论文（哪怕是中文核心）
  4. 参与大创项目并争取成为骨干

**如果你现在大三：**
- 目标：冲刺名头响亮的实习
- 行动：
  1. 立即申请华为勇敢星、京东TGT
  2. 准备腾讯犀牛鸟精英计划（次年2月申请）
  3. 整理研究成果，争取发表论文

**如果你现在大四：**
- 目标：争取转正或冲刺校招
- 行动：
  1. 如果已有实习，争取转正
  2. 申请华为全球顶尖AI人才、荣耀雏鹰
  3. 准备春招/秋招

### 10.4 核心逻辑总结

**社会上看名头，但名头需要官方认证才能背书。**

这些"犀牛鸟"、"勇敢星"、"TGT"、"雏鹰"都是大厂注册的人才品牌，比你自己说"我很厉害"有说服力得多。

**选择门槛相对低的先进入体系，再逐步升级名头，是本科生最务实的路径。**

**推荐路径：**
```
腾讯犀牛鸟开源（拿证书） → 华为勇敢星/京东TGT（实习） → 转正/校招冲刺更高名头
```
