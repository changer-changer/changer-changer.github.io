几何意义：函数 F 的等值面（如球面 x^2+y^2+z^2=R^2）

法向量（最核心公式）：
\boldsymbol{n} = \nabla F = \left(\frac{\partial F}{\partial x}, \frac{\partial F}{\partial y}, \frac{\partial F}{\partial z}\right)

单位法向量：\boldsymbol{n}^0 = \frac{\nabla F}{|\nabla F|}

切平面方程（点 P_0(x_0,y_0,z_0)）：
F_x(x_0,y_0,z_0)(x-x_0) + F_y(y-y_0) + F_z(z-z_0) = 0

2. 显式表示（函数图像形式）
形式：z = f(x,y)

转化：令 F(x,y,z) = f(x,y) - z = 0

法向量：
\boldsymbol{n} = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, -1\right) \quad \text{或} \quad (-f_x, -f_y, 1)

注意：(-f_x, -f_y, 1) 指向上方（z分量为正），(f_x, f_y, -1) 指向下方

切平面方程：
z - z_0 = f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)

---

3. 参数表示（双参数形式）
形式：\boldsymbol{r}(u,v) = (x(u,v), y(u,v), z(u,v))

几何意义：用两个独立参数 u,v 扫描整张曲面（如地球仪用经纬度）

切向量（曲面上两条坐标曲线）：
- \boldsymbol{r}u = \frac{\partial \boldsymbol{r}}{\partial u} = \left(\frac{\partial x}{\partial u}, \frac{\partial y}{\partial u}, \frac{\partial z}{\partial u}\right)（v固定，u变化时的切向）
- \boldsymbol{r}v = \frac{\partial \boldsymbol{r}}{\partial v} = \left(\frac{\partial x}{\partial v}, \frac{\partial y}{\partial v}, \frac{\partial z}{\partial v}\right)（u固定，v变化时的切向）

法向量（叉乘！）：
\boldsymbol{n} = \boldsymbol{r}u \times \boldsymbol{r}v

计算行列式：
\boldsymbol{r}u \times \boldsymbol{r}v = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\ x_u & y_u & z_u \\ x_v & y_v & z_v \end{vmatrix}

切平面方程（点法式，用 \boldsymbol{r}u \times \boldsymbol{r}v 作为法向量）：
(\boldsymbol{r} - \boldsymbol{r}(u_0,v_0)) \cdot (\boldsymbol{r}u \times \boldsymbol{r}v) = 0

二、空间曲线的表示方法及切向量计算

1. 单参数表示（轨迹形式）
形式：\boldsymbol{r}(t) = (x(t), y(t), z(t))

切向量：
\boldsymbol{T} = \boldsymbol{r}'(t) = (x'(t), y'(t), z'(t))

单位切向量：\boldsymbol{T}^0 = \frac{\boldsymbol{r}'(t)}{|\boldsymbol{r}'(t)|}

切线方程（点向式）：
\frac{x-x_0}{x'(t_0)} = \frac{y-y_0}{y'(t_0)} = \frac{z-z_0}{z'(t_0)}

法平面（垂直于切线的平面）：
x'(t_0)(x-x_0) + y'(t_0)(y-y_0) + z'(t_0)(z-z_0) = 0

---

2. 隐式表示（两曲面交线）
形式：
\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}

几何意义：两个曲面的交线，同时垂直于两个曲面的法向量

切向量（叉乘！）：
\boldsymbol{T} = \nabla F \times \nabla G = \begin{vmatrix} \boldsymbol{i} & \boldsymbol{j} & \boldsymbol{k} \\ F_x & F_y & F_z \\ G_x & G_y & G_z \end{vmatrix}

原理：切线同时在两个曲面上，故垂直于两曲面法向量，方向为两法向量叉乘。

切线方程：用上述 \boldsymbol{T} 作为方向向量，点向式写出。

四、法向量/切向量计算速查卡

曲面法向量（垂直于切平面）

```
隐式 F(x,y,z)=0  →  ∇F = (Fx, Fy, Fz)
显式 z=f(x,y)    →  (-fx, -fy, 1) 或 (fx, fy, -1)
参数 r(u,v)      →  ru × rv （叉乘！）
```

曲线切向量（沿曲线方向）

```
参数 r(t)        →  r'(t) （求导！）
交线 {F=0,G=0}   →  ∇F × ∇G （叉乘！）
```

五、核心几何关系图

```
空间几何对象
├─ 曲面 S
│  ├─ 法向量 n （垂直于曲面）
│  ├─ 切平面 Π （含所有切线）
│  └─ 表示方法
│     ├─ 隐式：F=0 → n=∇F
│     ├─ 显式：z=f(x,y) → n=(-fx,-fy,1)
│     └─ 参数：r(u,v) → n=ru×rv
│
└─ 曲线 C
   ├─ 切向量 T （沿曲线方向）
   ├─ 切线 l （过点沿T方向）
   ├─ 法平面 （垂直于切线）
   └─ 表示方法
      ├─ 参数：r(t) → T=r'(t)
      └─ 交线：{F=0,G=0} → T=∇F×∇G

关键关系：
- 曲面法向量 ⊥ 切平面内所有向量
- 曲线切向量 ⊥ 法平面
- 交线切向量 ⊥ 两曲面法向量（故为叉乘）
- 参数曲面法向量 ⊥ 两参数切向量（故为叉乘）
```

六、易错点警示

1. 叉乘顺序：\boldsymbol{r}u \times \boldsymbol{r}v 与 \boldsymbol{r}v \times \boldsymbol{r}u 方向相反（差负号），一般取使z分量为正的那个。

2. 显式曲面法向方向：(-f_x,-f_y,1) 指向上方（z正方向），(f_x,f_y,-1) 指向下方，根据题目"外法向"或"上侧"要求选择。

3. 单位化时机：先写出法向量/切向量，确认方向正确后再单位化（除以模长），不要过早单位化导致计算复杂。

4. 隐函数组求导：对于曲线 \{F=0,G=0\}，也可以用隐函数组求导法解出 dy/dx, dz/dx，则切向量为 (1, dy/dx, dz/dx)，与叉乘法结果一致。
