一、命题逻辑（核心基础）

1.1 基本联结词与优先级

联结词	符号	优先级	记忆口诀	
否定	¬	1（最高）	"非"最先算	
合取	∧	2	"且"次之	
析取	∨	3	"或"再次	
蕴含	→	4	"如果那么"	
等价	↔	5（最低）	"当且仅当"	

考试陷阱：P \rightarrow Q \rightarrow R 是 (P \rightarrow Q) \rightarrow R，不是 P \rightarrow (Q \rightarrow R)

---

1.2 等值演算（必考大题）

核心公式表（必须倒背如流）：

```
双重否定：¬¬A ≡ A
幂等律：   A∨A ≡ A,  A∧A ≡ A
交换律：   A∨B ≡ B∨A,  A∧B ≡ B∧A
结合律：   (A∨B)∨C ≡ A∨(B∨C),  (A∧B)∧C ≡ A∧(B∧C)
分配律：   A∧(B∨C) ≡ (A∧B)∨(A∧C)  ← 重点
          A∨(B∧C) ≡ (A∨B)∧(A∨C)  ← 重点
德摩根：   ¬(A∨B) ≡ ¬A∧¬B,  ¬(A∧B) ≡ ¬A∨¬B  ← 必考
吸收律：   A∨(A∧B) ≡ A,  A∧(A∨B) ≡ A  ← 快速化简
零一律：   A∨1 ≡ 1,  A∧0 ≡ 0,  A∨0 ≡ A,  A∧1 ≡ A
矛盾/排中：A∧¬A ≡ 0,  A∨¬A ≡ 1
蕴含等价：A→B ≡ ¬A∨B  ← 看到→马上转∨
双条件：  A↔B ≡ (A→B)∧(B→A) ≡ (A∧B)∨(¬A∧¬B)
假言易位：A→B ≡ ¬B→¬A
归谬论：  (A→B)∧(A→¬B) ≡ ¬A
```

1.3 范式理论（考试重点）

析取范式（DNF）与合取范式（CNF）

范式	形式	子句要求	
DNF	A_1 \lor A_2 \lor \cdots \lor A_n	每个A_i是简单合取式（文字的∧）	
CNF	B_1 \land B_2 \land \cdots \land B_n	每个B_i是简单析取式（文字的∨）	

求范式三步法：
1. 消去 → 和 ↔（用A \rightarrow B \equiv \neg A \lor B）
2. 否定深入（用德摩根律，使¬只作用于变元）
3. 分配律展开（求DNF用∧对∨分配，求CNF用∨对∧分配）

---

极小项与极大项（核心中的核心）

	极小项 m_i	极大项 M_i	
形式	n个文字的合取（∧）	n个文字的析取（∨）	
编号规则	原形=1，否定=0	原形=0，否定=1	
为真条件	只有赋值i使其为真	只有赋值i使其为假	
互补关系	m_i \equiv \neg M_i	M_i \equiv \neg m_i	

3变元示例：

赋值	m_i	M_i	
000	m_0 = \neg P \land \neg Q \land \neg R	M_0 = P \lor Q \lor R	
011	m_3 = \neg P \land Q \land R	M_3 = P \lor \neg Q \lor \neg R	
111	m_7 = P \land Q \land R	M_7 = \neg P \lor \neg Q \lor \neg R	

---

主范式（唯一标准形）

主范式	构成	包含的项	唯一性	
主析取范式(PDNF)	极小项的∨	使公式为真的所有赋值	✓ 唯一	
主合取范式(PCNF)	极大项的∧	使公式为假的所有赋值	✓ 唯一	

互补关系：
- PDNF有k个极小项 → PCNF有2^n - k个极大项
- 下标集合互补：若PDNF = \{0,1,3,5,7\}，则PCNF = \{2,4,6\}

1.4 题型一：等值证明（永真式证明）

方法：等价演算法（推荐）或真值表法

标准格式：

```
证明：A ≡ [第一步，用××律]
      ≡ [第二步，用××律]
      ...
      ≡ 1（或T，或永真式）
∴ 原式是永真式。
```

关键：每步注明公式名称，严禁跳步。

---

1.5 题型二：求主范式

方法一：等值演算法
1. 先求DNF/CNF
2. 对每个子句，缺什么变量就补什么（用X \lor \neg X \equiv 1或X \land \neg X \equiv 0）
3. 展开后用幂等律去重，按编号排序

方法二：真值表法（变量≤3时推荐）
1. 列出所有2^n个赋值
2. 找出使公式为真的赋值 → 对应极小项 → PDNF
3. 找出使公式为假的赋值 → 对应极大项 → PCNF

---

1.6 题型三：给定方向的方向导数/梯度问题

虽然这是高数内容，但逻辑结构与离散数学类似：
- 梯度 = 各偏导数组成的"向量"
- 方向导数 = 梯度与方向单位向量的"点乘"（内积）
- 最大方向导数 = 梯度方向，值为梯度模长

2.1 基本概念

概念	符号	含义

2.2 量词否定等值式（重点）

\neg \forall x A(x) \equiv \exists x \neg A(x)
\neg \exists x A(x) \equiv \forall x \neg A(x)

口诀："否定跨越量词，∀变∃，∃变∀"

三、集合论基础

3.1 基本运算

运算	定义	文氏图特征	
并 A \cup B	\{x \mid x \in A \lor x \in B\}	两圆全部	
交 A \cap B	\{x \mid x \in A \land x \in B\}	两圆重叠	
差 A - B	\{x \mid x \in A \land x \notin B\}	A圆去掉重叠	
补 \overline{A}	\{x \mid x \notin A\}	全集中A外	

3.2 容斥原理

|A \cup B| = |A| + |B| - |A \cap B|

|A \cup B \cup C| = |A| + |B| + |C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|

四、关系与函数

4.1 关系的性质

性质	定义	矩阵特征	
自反	\forall x, (x,x) \in R	主对角线全1	
反自反	\forall x, (x,x) \notin R	主对角线全0	
对称	(x,y) \in R \rightarrow (y,x) \in R	矩阵对称	
反对称	(x,y) \in R \land (y,x) \in R \rightarrow x=y	对称位置不同时为1	
传递	(x,y) \in R \land (y,z) \in R \rightarrow (x,z) \in R	R^2 \subseteq R	

4.2 等价关系与划分

等价关系 = 自反 + 对称 + 传递

等价类：[a]R = \{x \mid (a,x) \in R\}

商集：A/R = \{[a]R \mid a \in A\}（所有等价类的集合）

五、图论基础（如有涉及）

5.1 基本概念

- 握手定理：\sum{v \in V} d(v) = 2|E|（所有顶点度数之和 = 2倍边数）
- 欧拉回路：经过每条边恰好一次的回路 ↔ 连通且所有顶点度数为偶数

6.2 常见失分点

错误类型	正确做法	
忘记单位化方向向量	方向导数必须除以	
混淆m_i和M_i编号规则	极小项：原形=1；极大项：原形=0	
求二阶偏导时f_1结构不清	f_1仍是三变量函数，继续链式法则	
德摩根律用错	\neg(A \land B) = \neg A \lor \neg B，不是\neg A \land \neg B	
蕴含等价记错	A \rightarrow B = \neg A \lor B，不是\neg(A \lor B)
