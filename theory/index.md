## 核心术语

| 术语 | 定义 |
|------|------|
| **趋向性** | 性质即趋向。万物在时空中沿逻辑方向运动的内在属性 |
| **合理性** | 宇宙的底层逻辑结构。一切存在皆合理 |
| **复杂性富集** | 生命通过吸收负熵，在局部降低熵增的过程 |
| **具身组块** | 身体通过实践建立的神经回路，是能力的物质基础 |
| **模态** | 大脑对世界的映射模型。训练越完整，预测越准确 |
| **世界线** | 一个物体在时空中的完整轨迹 |
| **本征趋向** | 物质结构最底层的运动方向，如"活下去" |
